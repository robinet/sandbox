# Sandbox Repository

A **PERSONAL** collection of scripts and tools for automation and development tasks.

## Overview

This repository contains various scripts and tools to streamline development workflows and automate repetitive tasks.

## Repository Structure

```plaintext
.
├── docs/                 # Documentation files
├── src/                  # Source code (empty placeholder)
├── tools/                # Main tools directory
│   ├── github/           # GitHub-related scripts
│   ├── hashicorp/        # HashiCorp tools installation scripts
│   ├── windows-terminal/ # Windows Terminal configuration scripts
│   └── Shared.ps1        # Shared PowerShell functions
├── .gitignore            # Git ignore file
└── README.md             # This file
```

## Tools

### GitHub Tools

- Get-GitHubRepositories.ps1 - Get GitHub repositories
- Install-GitHubRelease.ps1 - Install GitHub releases
- Install.ps1 - Install GitHub tools
- Set-GitHubConnection.ps1 - Set up GitHub connection

### HashiCorp Tools

- Install-HashicorpRelease.ps1 - Install HashiCorp releases

### Windows Terminal

- Get-TerminalGitHubCommandsFromFolder.ps1 - Get GitHub commands from folder

## Getting Started

1. Clone this repository
2. Ensure you have PowerShell 5.1 or later
3. Run scripts as needed
